ArHub Icon Library

Professional SVG icons for modern web development

Installation

Add this CSS file to your HTML to use the icons:

<link rel="stylesheet" href="https://arlabs07.github.io/Arhub07.github.io/arhub-icons.css">

Then use any icon with the class format: <i class="ar-icon-name"></i>
