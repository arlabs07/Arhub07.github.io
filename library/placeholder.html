üîó API Usage

Generate placeholder images by using URL parameters:

https://arlabs07.github.io/Arhub07.github.io/library/placeholder.html?width=300&height=200&bg=blue&text=Hello&color=white

URL Parameters:

  • width - Image width (default: 300)
  • height - Image height (default: 200)
  • bg - Background color (hex, named color, or "gradient")
  • color - Text color (default: white)
  • text - Custom text (default: dimensions)
  • format - Image format: png, jpg, webp (default: png)

Example URLs:

?width=400&height=300&bg=ff6b6b&text=Placeholder
?width=500&height=400&bg=gradient&color=black&text=Beautiful Image
?width=600&height=200&bg=2196f3&color=white&format=jpg
